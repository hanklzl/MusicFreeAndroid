- [ ] **Step 1.3: 在 9 个模块各自 build.gradle.kts 中添加 Kotlin toolchain + jvmTarget 配置**

> 设计取舍：toolchain（JDK 25）与字节码目标（Java 17）必须分别设置：
> - `kotlin { jvmToolchain(25) }` — 用 JDK 25 跑编译进程
> - `kotlin { compilerOptions { jvmTarget.set(JvmTarget.JVM_17) } }` — Kotlin 输出 Java 17 字节码（不显式设的话默认跟随 toolchain = 25，与 Android 不兼容）
> - `compileOptions { source/targetCompatibility = VERSION_17 }` — Java 源也输出 Java 17 字节码
>
> 三处必须同步在 17，否则 R8 / dexer 会报 bytecode level 不一致。

在每个模块的 `build.gradle.kts` **`android { ... }` 块内**改 `compileOptions`：

```kotlin
    compileOptions {
        sourceCompatibility = JavaVersion.VERSION_17
        targetCompatibility = JavaVersion.VERSION_17
    }
```

并在 **`android { ... }` 块外、`dependencies { ... }` 块前**追加：

```kotlin
kotlin {
    jvmToolchain(25)
    compilerOptions {
        jvmTarget.set(org.jetbrains.kotlin.gradle.dsl.JvmTarget.JVM_17)
    }
}
```

OkHttp 5 已知的 source-break（最常见）：

| 旧（OkHttp 4）                                | 新（OkHttp 5）                                       |
| --------------------------------------------- | --------------------------------------------------- |
| `MediaType.parse("...")`                       | `"...".toMediaType()`                               |
| `MediaType.parse("...").toMediaTypeOrNull()` | `"...".toMediaTypeOrNull()`                         |
| `RequestBody.create(MediaType, String)`        | `String.toRequestBody(MediaType)`                   |
| `RequestBody.create(MediaType, ByteArray)`     | `ByteArray.toRequestBody(MediaType)`                |
| `Headers.of(map)` (Java-style)                 | `map.toHeaders()` (Kotlin extension)                |
| `HttpUrl.parse(url)`                           | `url.toHttpUrlOrNull()` 或 `url.toHttpUrl()`        |
| `okhttp3.coroutines.executeAsync()`            | OkHttp 5 内置 `Call.executeAsync()`，包路径变化     |
| `Cache(File, Long)` 构造                       | `Cache(directory: File, maxSize: Long)` 命名参数    |

依赖的 import 通常需要新增：
```kotlin
import okhttp3.MediaType.Companion.toMediaType
import okhttp3.MediaType.Companion.toMediaTypeOrNull
import okhttp3.RequestBody.Companion.toRequestBody
import okhttp3.HttpUrl.Companion.toHttpUrl
import okhttp3.Headers.Companion.toHeaders
```
