技术要点：
  • SearchViewModel 重构: 提交搜索时并行调用所有可搜索插件，结果按 Map<String/*platform*/, SearchUiState> 存储
  • 第一层 Tab: ScrollableTabRow — 单曲/专辑/歌手/歌单（对应 RN supportedSearchType）
  • 第二层 Tab: ScrollableTabRow（无指示器）— 各插件源名称，按 PluginManager 排序
  • 搜索历史: DataStore 存储，最多保留 20 条，去重 + 最新置顶
  • 三点菜单: 复用 LocalScreen 的 AddToPlaylistDialog，后续扩展为完整选项面板
  • 搜索栏: 去掉 TopAppBar，自定义 Row（返回键 + 药丸输入框 + 搜索按钮），高度 rpx(88)

技术要点：
  • 高斯模糊: Modifier.blur(50.dp, edgeTreatment = BlurredEdgeTreatment.Unbounded) + alpha(0.5f)，底层纯黑 Box
  • 顶部栏居中: 去掉 Material3 TopAppBar，自定义 Row + 中间 Column(Arrangement.Center, Alignment.CenterHorizontally)，高 rpx(150)
  • 封面尺寸: 竖屏 rpx(500)×rpx(500)
  • 操作栏: 6 个图标按钮行，高 rpx(80)（收藏/品质/下载/倍速/评论/更多）— 本次先做 UI 占位，功能后续接入
  • 控制按钮: 中心播放键 rpx(96)，两侧 rpx(56)
  • SeekBar: 白灰配色（#cccccc / #999999 / thumb #dddddd）

三、MiniPlayer 修复

  • 封面图加载: MiniPlayerContent 中用 AsyncImage(model = uiModel.coverUri) 替换 MusicNote 占位符，圆形裁剪 clip(CircleShape)
  • 首页统一: MainActivity 中去掉 mock MiniPlayer 分支，所有页面（除 PlayerRoute）统一使用真实 MiniPlayer
  • 尺寸: 高 rpx(132)，封面 rpx(96)×rpx(96) 圆形

四、健壮性修复

  • URL 空值防护: toMediaItem() 中 URL 为空时抛出明确异常或返回 error state，不静默跳过
  • 连接失败反馈: withConnectedController 失败时通过 SharedFlow 发送 Toast 事件
  • 封面图诊断: 运行时日志记录 artwork URL，Coil 添加自定义 Headers（Referer）以防资源服务器拦截
