# Logan Decode Tooling

Use this directory to decode feedback bundles generated by the app.

Debug logs are encrypted with the development key configured in `app/build.gradle.kts`.

Release logs require environment variables for the release key and IV:

```bash
export LOGAN_AES_KEY=your_release_key_16
export LOGAN_AES_IV=your_release_iv_16
```

Run:

```bash
tools/logan/decode-logan.sh path/to/musicfree-feedback.zip
```

The script accepts either a feedback zip package or a directory containing raw
`logan/` files. It compiles the local JDK decoder, decrypts and decompresses
each raw Logan file, and writes text output to `tools/logan/out/decoded/*.txt`
by default. Pass a second argument such as `smoke` to write under
`tools/logan/out/smoke/`.

Requirements: `javac`, `java`, and `unzip` when decoding a zip package. Secrets
must never be written to source files or logs.
